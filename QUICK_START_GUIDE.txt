===========================================
PHARMACY WEB APP SYSTEM - QUICK START GUIDE
===========================================

📱 FLUTTER MOBILE APP + 🌐 WEB SYSTEM
Complete Pharmacy Management System

===========================================
PREREQUISITES
===========================================

1. XAMPP (for web system)
   - Download: https://www.apachefriends.org/
   - Install and start Apache + MySQL

2. Flutter SDK (for mobile app)
   - Download: https://flutter.dev/docs/get-started/install
   - Add to PATH
   - Run: flutter doctor

3. Android Studio / VS Code (for Flutter development)

===========================================
STEP 1: SETUP WEB SYSTEM (PHP + MySQL)
===========================================

1. Start XAMPP:
   - Open XAMPP Control Panel
   - Start Apache and MySQL services

2. Setup Database:
   - Open: http://localhost/phpmyadmin
   - Create database: pharmacy_db
   - Import: database/pharmacy_db.sql

3. Configure Database Connection:
   - Edit: config/database.php
   - Update hostname, username, password if needed

4. Test Web System:
   - Open: http://localhost/pharmacy_web_app_system
   - Should show login page

===========================================
STEP 2: SETUP FLUTTER MOBILE APP
===========================================

1. Navigate to Flutter App:
   cd pharmacy_app

2. Install Dependencies:
   flutter pub get

3. Run the App:
   flutter run

   OR for specific platform:
   flutter run -d chrome    (Web)
   flutter run -d android   (Android)
   flutter run -d windows   (Windows)

===========================================
STEP 3: TEST THE SYSTEM
===========================================

🌐 WEB SYSTEM TESTING:
1. Open: http://localhost/pharmacy_web_app_system
2. Login with existing users or create new ones
3. Test all roles: Admin, Pharmacy, Customer

📱 FLUTTER APP TESTING:
1. App starts at login screen
2. Login with same credentials as web system
3. Navigate through all pages for each role

===========================================
TROUBLESHOOTING
===========================================

❌ Web System Not Working:
- Check XAMPP is running
- Verify database connection in config/database.php
- Check Apache port (default: 80)

❌ Flutter App Not Working:
- Run: flutter doctor
- Check dependencies: flutter pub get
- Verify API endpoints in lib/services/api_service.dart

❌ Database Connection Issues:
- Verify MySQL is running in XAMPP
- Check database name: pharmacy_db
- Test connection in phpMyAdmin

❌ File Upload Not Working:
- Check uploads/prescriptions/ folder permissions
- Verify file size limits in PHP

===========================================
API ENDPOINTS (for reference)
===========================================

Base URL: http://localhost/pharmacy_web_app_system/api/

Authentication:
- POST /login.php
- POST /signup.php

Users:
- GET/POST/PUT/DELETE /users.php

Medicines:
- GET/POST/PUT/DELETE /medicines.php

Orders:
- GET/POST/PUT/DELETE /orders.php

Prescriptions:
- GET/POST/PUT/DELETE /prescriptions.php

File Upload:
- POST /upload_prescription.php
