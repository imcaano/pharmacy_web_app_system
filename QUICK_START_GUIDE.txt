PREREQUISITES
===========================================

1. XAMPP (for web system)
   - Download: https://www.apachefriends.org/
   - Install and start Apache + MySQL

2. Flutter SDK (for mobile app)
   - Download: https://flutter.dev/docs/get-started/install
   - Add to PATH
   - Run: flutter doctor

3. Android Studio / VS Code (for Flutter development)

===========================================
STEP 1: SETUP WEB SYSTEM (PHP + MySQL)
===========================================

1. Start XAMPP:
   - Open XAMPP Control Panel
   - Start Apache and MySQL services

2. Setup Database:
   - Open: http://localhost/phpmyadmin
   - Create database: pharmacy_db
   - Import: database/pharmacy_db.sql

3. Configure Database Connection:
   - Edit: config/database.php
   - Update hostname, username, password if needed

4. Test Web System:
   - Open: http://localhost/pharmacy_web_app_system
   - Should show login page

===========================================
STEP 2: SETUP FLUTTER MOBILE APP
===========================================

1. Navigate to Flutter App:
   cd pharmacy_app

2. Install Dependencies:
   flutter pub get

3. Run the App:
   flutter run

   OR for specific platform:
   flutter run -d chrome    (Web)
   flutter run -d android   (Android)
   flutter run -d windows   (Windows)
